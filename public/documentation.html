myFlix

myFlix provides users with access to information about different movies, directors, and genres. They are able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by name /movies/[Title] GET None A JSON object holding data about a single movie, containing name, and classes property. Example: { name: “The Godfather”; classes: { description: xxx; genre: xxx; director: xxx; img URL: xxx; featured: xxx; } }
Get data about a genre by name /movies/genres/[Name] GET None A JSON object holding data about a single genre. Example: { name: “Drama”; description: xxx; }
Get data about a director by name /movies/directors/[Name] GET None A JSON object holding data about a director, containing name, and classes property. Example: { name: “Francis Ford Coppola”; classes: { bio: xxx; birth year: xxx; death year: xxx; } }
Add a user /users POST A JSON object holding data about the new user. Example: { name: “John Doe”; classes: { username: xxx; password: xxx; e-mail: xxx; birthday: xxx; fovarites: xxx; } } A JSON object holding data about the new user that was added, containing an ID. Example: { id: 5; name: “John Doe”; classes: { username: xxx; password: xxx; e-mail: xxx; birthday: xxx; fovarites: xxx; } }
